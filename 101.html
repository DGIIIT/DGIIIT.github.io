Getting started 101

It's a lot of work to create a professional website, so if you're new to web development, we encourage you to start small. You won't build another Facebook right away, but it's not hard to get your own simple website online, so we'll start there.


Before starting it's better to you to know how web actually works, please watch the below playlist(optional) so just to make sure we are on the same page(metaphorically)





When it comes to tools for building a website, there's a lot to pick from. If you're just starting out, you might be confused by the array of code editors, frameworks, and testing tools out there. Let me help you choose to begin some basic web development.
